> 
> 2. It's light weight, easy to use and provides real-time data with sub-7ms latency.
>
> ### How to use? (TUTORIAL FOR WINDOWS)
> 1. Install OpenJDK 22 or newer, you can download OpenJDK on [Java 25](https://jdk.java.net/25/)
>
> 2. Once it downloads, extract the .zip file. One the jdk-25 or newer files select "cut".
>
> 3. Go to your C: drive, open (Program Files).
>
> 4. Make a new folder called Java, paste your jdk-25 file inside.
>
> 5. Go to the Windows Search bar and type (Edit the system environment variables). Press on (Envrionment Variables)
>
> 6. On System variables, press New. Variable Name: JAVA_HOME Varaible Value: paste your Java folder from Program Files
>
> 7. Double click Path, Press new. Type %JAVA_HOME%bin
>
> 8. Open Command Prompt and type (java --help) to see if you downloaded it correctly
>
> 9. If everything worked correctly. Download the .jar file in releases. Find it in downloads and select copy as path.
>
> 10. Go to Command Prompt and type java -jar (COPY_YOUR_JAR_PATH)
